# FinanceTracker — Учёт личных финансов

Настольное приложение для учёта доходов и расходов, планирования бюджета
и анализа трат по категориям. Разработано на платформе .NET с использованием
технологии Windows Forms.

## Функции

- Ведение операций (доходы / расходы) с привязкой к категориям
- Фильтрация операций по дате, типу, категории и ключевым словам
- Планирование бюджета (лимиты расходов по категориям на период)
- Формирование выписок в форматах CSV, HTML и Markdown
- Визуализация расходов по категориям (круговая диаграмма GDI+)
- Отмена и повтор операций (паттерн «Команда»)

## Архитектура

| Слой       | Папка        | Описание                                     |
|------------|--------------|----------------------------------------------|
| Модель     | `Model/`     | Классы предметной области, перечисления       |
| Хранилище  | `Store/`     | Файловое хранение JSON, таблицы, БД-одиночка |
| Службы     | `Services/`  | Бизнес-логика, фильтрация, выгрузки, отмена  |
| Интерфейс  | `Forms/`     | Формы Windows Forms                           |

## Паттерны проектирования

- **Наблюдатель** (Observer) — событие `Mutated` таблицы → `Changed` в FinanceDb → обновление форм
- **Стратегия** (Strategy) — `IOperationFilter` + `FilterChain` (PeriodFilter, KindFilter, CategoryFilter, TextFilter)
- **Фабричный метод** (Factory Method) — `StatementFactory.Create()` → `CsvStatement` / `HtmlStatement` / `MarkdownStatement`
- **Команда** (Command) — `IUndoableAction` + `LambdaAction` + `ActionStack` (отмена / повтор)
